# SocGen Sentinel
> **AI-Powered Third-Party Vendor Risk Intelligence Platform**

SocGen Sentinel is an enterprise-grade cybersecurity and banking dashboard designed to continuously monitor, score, and analyze third-party vendor risk vectors. Built for financial institutions, it replaces manual sheets with a centralized intelligence hub that integrates compliance tracking, contract NLP legal parsing, and dynamic threat simulation modeling.

---

## 🚀 Key Features

* **Centralized Vendor Registry**: Maintain profiles of downstream dependencies, data access limits, subprocessors count, and shared assets.
* **Continuous Risk Scoring Engine**: Calculates risk indices (Low, Medium, High, Critical) using active breaches, data sensitivity, and compliance posture features.
* **Compliance Auditing Tracker**: Aggregates SOC2, ISO27001, and GDPR alignments, flagging missing documents and expiring compliance certificates.
* **AI SLA Contract Parser**: Scans vendor agreements to extract indemnity limits, breach notification timelines, and privacy clauses, highlighting risk exposures automatically.
* **Risk Intelligence Sandbox**: Run What-If simulation modeling (e.g., enforcing MFA, encrypting backups) to estimate mitigated risk outcomes prior to implementation.
* **Threat Stream Activity Feed**: Real-time notifications and timeline audit logs tracking dark web exposure leaks and zero-day patches.

## 🛠️ Technology Stack

* **Frontend**: React (Vite)
* **Styling**: Tailwind CSS v4
* **Animations**: Framer Motion
* **Visualizations**: Recharts
* **Icons**: Lucide React

---

## 🎨 Design Theme

* **Dark Mode**: High-contrast dark navy space background overlayed with a generated high-tech visual HUD texture.
* **Glassmorphism Layout**: Translucent panel cards with radial glows and pulsing status indicators.
* **Regulatory Indicators**: Harmonic color indicators (Red for critical, Orange for high warnings, Green for compliant).

## 💻 Getting Started

### Prerequisites
* **Node.js** (v18 or higher)
* **npm**

### Installation & Run

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser to interact with the platform.
